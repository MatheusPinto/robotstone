◆ StartDistributedExperiment()

void RobotMaster::StartDistributedExperiment ( void  )

Start the experiment related to the processing and communication domain and run until experiment finishes.

After experiment initialization, this function will be the management task superloop, that will start the synthetic tasks, get their experiment results and re-run then case necessary or finish the experiment in Slave node.
